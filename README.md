# AI-Prompt-Quality-Evaluation-Clarification-System
Designed and implemented an LLM-driven prompt evaluation pipeline that classifies user inputs by quality, triggers automated clarification flows for ambiguous or incomplete prompts, and synthesizes enriched prompts by merging user responses with original inputs to optimize LLM performance
Here is a **clear, professional, and project-ready README.md** for your
**AI-Prompt-Quality-Evaluation-Clarification-System** repository.



## 🚀 Project Purpose

* Detect **unclear, vague, or low-quality prompts**
* Provide **structured clarification questions**
* Improve prompt input before sending it to an AI system
* Experiment with prompt engineering best practices
* Build a foundation for **production-ready AI prompt pipelines**

## ⚙️ Setup & Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/mostahidhasan2019331070/AI-Prompt-Quality-Evaluation-Clarification-System.git
cd AI-Prompt-Quality-Evaluation-Clarification-System
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Configure environment variables

Create a `.env.local` file (already included for local use):

```env
# example
VITE_APP_NAME=PromptEvaluationSystem
```

> ⚠️ Do not commit sensitive keys to public repositories.

---

## ▶️ Run the Project

```bash
npm run dev
```

The app will start locally using **Vite**.
